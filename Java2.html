Arrays and the const variable

The const variable

The 'const' variable or the constant variable

  • Variables defined with the 'const' cannot be Redeclared
  • Variables defined with the 'const' cannot be Reassined
  • Variables defined with the 'const' cannot be have Block Scope
The better use of the 'const' variable is when you declare
  • A new Array
  • A new Object
  • A new Function
  • A new RegExp
